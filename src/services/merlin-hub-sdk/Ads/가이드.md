# 📢 Ads (광고 및 반응형 윙) SDK 가이드

허브 생태계 내 앱들의 수익화 및 앱 설치/공유 유도를 담당하는 6번째 핵심 SDK 모듈입니다.
사이드 윙(데스크톱)과 하단 고정 배너(모바일)를 각각 따로 렌더링하지 않고, 단일 컴포넌트인 `HubResponsiveWing`을 통해 일괄 제어합니다. 구글 애드센스와도 완벽하게 연동됩니다.

## 1. 컴포넌트

### 1) `HubResponsiveWing`
- **역할**: 1120px 이상의 화면에서는 우측 날개(`aside`)로, 미만의 화면에서는 화면 하단 스티키 배너로 자동 스위칭되는 반응형 레이아웃입니다.
- **주요 Props**:
  - `bannerId` (필수): 하단 배너를 사용자가 'X' 버튼으로 닫았을 때 `localStorage`에 상태를 저장하기 위한 고유 식별자.
  - `hide`: 광고 프리패스(adFree) 계정이거나 결제/관리자 페이지에서 광고 영역 전체를 숨기고 싶을 때 `true`를 전달합니다.
  - `shareTitle`, `shareDescription`: 이 값을 전달하면 우측 윙 최상단에 **공유 카드(`HubShareSquare`)**가 자동으로 생성됩니다.
  - `adClient`, `sideAdSlot`, `bottomAdSlot`: 구글 애드센스 계정 정보. 이것을 전달하면 빈 자리에 실제 구글 광고 스크립트가 로드됩니다.

### 2) `HubGoogleAd`
- **역할**: Next.js 환경에서 CSR 방식으로 구글 광고(`adsbygoogle`)를 안전하게 렌더링해 주는 래퍼 컴포넌트입니다.
- 단독으로 사용할 수도 있지만 보통 `HubResponsiveWing` 내부에서 알아서 호출되므로 직접 사용할 일은 적습니다.

## 2. 사용 예시

```tsx
import { HubResponsiveWing } from "@/src/services/merlin-hub-sdk/react"

export default function Layout({ children }) {
  // adFree 등의 상태 감지
  const isAdFree = useAdFreeCheck();

  return (
    <div>
      {children}
      
      <HubResponsiveWing
        bannerId="myapp_v1"
        hide={isAdFree}
        shareTitle="식단 관리가 고민될 땐? 뭐먹지!"
        shareDescription="가족과 함께 식단을 투표해 보세요."
        adClient="ca-pub-123456789"
        sideAdSlot="111111"
        bottomAdSlot="222222"
      />
    </div>
  )
}
```

## 3. 핵심 규칙
- **하드코딩 배제**: 더 이상 앱 내부에 직접 `<aside>`나 하단 배너 태그를 직접 작성하지 말고, 이 SDK 하나로 퉁칩니다.
- **SSOT 적용**: 배너 닫기 상태는 SDK 내부에서 로컬 스토리지를 통해 알아서 유지됩니다.
