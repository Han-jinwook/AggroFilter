## 🔄 허브 인프라 변경 및 공지 (Hub Changelog)

- **2026-05-25**: **SDK v1.4.0 배포 / HubBenefitModal v1.0.0**. 무료 체험 유저 대상 혜택 안내 모달(`HubBenefitModal`) 및 지연 노출 트리거(`useBenefitTrigger`, `markFreeTrialCompleted`) 추가. 앱별 맞춤 혜택 설정 지원. 허브 서버에 `Vaulter`(금고지기) 메일 브랜딩 분기 지원 추가.- **2026-05-24**: **SDK v1.3.9 배포 / HubNotificationCard 개편**. 알림 설정 단순화(스마트 알림 단일화)를 위해 `HubNotificationCard`를 단일 토글 형태로 개편. `title`, `toggleLabel`, `description` 프로퍼티 주입을 통해 개별 패밀리 앱이 원하는 문구로 알림 동의 UI를 렌더링하고 상태를 제어할 수 있도록 보완.
- **2026-05-24**: **SDK v1.3.8 배포 / HubProfileCards 도입**. 프로필 설정 기능을 기존 모달에서 카드형 UI 3종(`HubProfileCard`, `HubNotificationCard`, `HubLogoutCard`)으로 전면 개편. 개별 앱 내 프로필 설정 페이지 페이지 임베딩 방식 지원.
- **2026-05-24**: **SDK v1.3.7 배포 / HubProfileModal v1.0.0**. 모든 패밀리 앱에서 공통으로 호출할 수 있는 통합 프로필 설정 모달 팝업 추가. 닉네임 수정 및 아바타 이미지 업로드, 통합 로그아웃 기능 내장.
- **2026-05-23**: **SDK v1.3.6 배포 / HubAuthModal v2.0.1**. HubAuthModal 폰트 크기 최적화(15px)로 긴 텍스트 잘림 현상 방지 및 이메일 입력 시 히스토리 내 동일 이메일 버튼 숨김(필터링) 처리하여 중복 UX 제거. 라이브러리 가이드에 SPA 프록시 브릿지 구성 강제 규정 및 Netlify `_redirects` 동적 생성 충돌 관련 트러블슈팅 안내 대폭 보강.
- **2026-05-23**: **SDK v1.3.5 배포 / HubPurchaseWidget v1.3.0**. 불필요한 잔액 박스를 제거하여 위젯 공간을 최적화하고, 서버 부하 감소를 위해 이용내역의 필터링(유형/앱/기간) 및 페이징 로직을 모두 프론트엔드(SDK) 단에서 자체 전담하도록 구조를 개편. 네이버페이 스타일의 직관적인 필터 UI 추가.
- **2026-05-23**: **동적 과금 한도 규격화 적용**. `family_apps` 내 과금 정책(`client_secret`)에 `min_price`(최저 요금 제한) 및 `max_price`(최대 요금 제한) 값을 파싱하여 요금을 클램핑하는 로직이 서버에 반영되었습니다. 어그로필터는 최대 100C(최저 10C)로 제한 한도를 규격 적용 완료.
- **2026-05-21**: **SDK v1.3.4 배포 / useHubPayment v1.3.4**. 최초 결제 요청 시 KCP 동적 로딩 스크립트(jQuery, kcp_spay_cross_hub.js 등)가 완료되지 않아 발생하던 무한 로딩 버그를, 예외 포착 후 자동 재시도하는 예외 복구(Retry) 로직 도입으로 근본적 해결 완료.
- **2026-05-20**: **SDK v1.3.4 배포 / HubPurchaseWidget v1.2.0**. 이용권 카드 UI 디자인을 원본 이미지(PNG/JPG) 형태로 원복 및 SDK assets 내부 통합 완료.
- **2026-05-19**: **SDK v1.3.0 배포**. 계좌이체(PABK) 결제 수단은 정부 공문 권고로 완전 제거되었으며, 휴대폰 결제는 심사 중으로 처리되었습니다. 추천인 보상은 초대자 500C(가입자 최초 기능 소모 시점에 지급), 가입자는 별도 추천인 보상 없이 기본 가입 축하 500C로 적용 완료. 어그로필터 충전 UI 연동 완료.
- **2026-05-19**: **SDK v1.2.2 배포**. 비회원 선(先) 체험 가불 정산 및 분석 정보 마이그레이션 기술 표준 제정 및 활용가이드 등재. 🐾💳
- **2026-05-18**: **SDK v1.2.1 배포**. 인증 모듈에서 불필요한 이메일 fallback 로직 완전 제거. JWT 내 진짜 UUID 기반으로만 통신하도록 교정 완료.
- **2026-05-18**: **SDK v1.2.0 배포**. 라이브러리 내 폴더 구조 전면 개편 (Auth, Wallet, Session, Referral, Notification, CoreLogic, UI 등 도메인별 분리). 유지보수성 극대화. 📁✨
- **2026-05-17**: **SDK v1.1.2 배포**. 프로필 수정 후 즉시 헤더 닉네임과 아바타가 변경되지 않던 Next.js fetch 캐싱 버그 해결. 🛠️⚡
