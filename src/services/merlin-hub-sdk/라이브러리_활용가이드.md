# 📖 멀린 SDK 라이브러리 활용 가이드 (v1.1.0+)

본 문서는 Merlin Hub SDK의 기술적인 사용 방법과 모든 모듈별 예제 코드를 관리합니다.

---

## 🛠️ 0. 기본 설정 (Setup)

1. **에셋 복사**: `허브_라이브러리/assets/`의 파일들을 개별 앱의 `public/hub_assets/` 폴더로 복사합니다.
2. **Core 라이브러리 연동**: `Core/` 폴더의 파일들을 앱의 `src/services/merlin-hub-sdk/` 경로에 배치합니다.

### 2단계: 에셋(이미지) 수동 배치 (필수)
부품들이 사용하는 공식 이미지들을 브라우저가 인식할 수 있도록 배치합니다.
1. 개별 앱의 `public/` 폴더 내에 `hub_assets/` 폴더를 생성합니다.
2. `src/services/merlin-hub-sdk/assets/` 내의 모든 파일을 `public/hub_assets/`로 복사합니다.
   - 예: `guest_paw.png` → `public/hub_assets/guest_paw.png`

### 3단계: 통합 배선 및 레이아웃 설정
1. **입구 이원화 (필수)**: 
   - **`@/src/services/merlin-hub-sdk`**: API Route, Server Component용 (Pure TS)
   - **`@/src/services/merlin-hub-sdk/react`**: UI, Hooks용 ('use client' 선언됨)
2. **알림 연동**: `app/layout.tsx` (또는 최상위 레이아웃)에 `<HubNotifier />`를 배치하여 시스템 전체 알림을 활성화합니다.

### 🏁 연동 완료 체크리스트 (v1.0.2+)
- [ ] **브랜딩**: 유저 노출 UI에서 "패밀리", "멀린", "허브" 등의 내부 용어가 제거되었는가?
- [ ] **에셋**: `public/hub_assets/guest_paw.png` 및 앱 전용 로고가 설정되었는가?
- [ ] **세션**: `useHubSession`에서 `profile.success`를 검사하는가?
- [ ] **추천**: 추천인 코드는 수동 입력 대신 공유 링크(`?ref=CODE`)를 통한 자동 인증을 권장함.

## 🧩 1. 프로필 위젯 세트 (Profile & Session)
- **부품**: `useHubSession` (Core), `HubProfileWidget` (Custom)
- **용도**: 게스트/회원 상태 표시 및 세션 관리.

---

## 🔒 2. 이메일 인증 통합 모듈 (Auth Suite)
- **부품**: `useHubAuth` (Core), `HubAuthModal` (Custom)
- **용도**: 6자리 OTP 기반 통합 로그인 및 브랜딩 강화.
- **v1.1.0 변경사항**: `requestOTP(email, appId)`를 통해 앱별 맞춤형 이메일 발송(발송자명, 제목)이 지원됩니다.

---

## 💳 3. 결제 및 코인 (Payment Suite)
- **부품**: `useHubPayment` (Core), `HubPaymentTrigger` (Custom)
- **용도**: KCP 결제창 호출 및 지갑 잔액 조회.

---

## 📬 4. 알림 브릿지 (Notification Bridge)

### [Core] `useHubNotifier` Hook (v1.0.0)
허브를 통해 이메일을 발송하고 유저의 알림 설정을 관리합니다.

```typescript
import { useHubNotifier } from '@/services/merlin-hub-sdk/Core/useHubNotifier';

const { sendEmail, updateSettings } = useHubNotifier();
```

---

## 🤝 5. 추천 및 보상 (Referral Suite)

### [Core] `useHubReferral` Hook (v1.0.0)
나의 추천 코드 조회 및 타인의 코드 등록을 처리합니다.

### [Custom] `HubReferralWidget` Component (v1.0.0)
친구 초대와 코드 복사를 한 번에 제공하는 세련된 그라데이션 위젯입니다.

```tsx
import { HubReferralWidget } from '@/services/merlin-hub-sdk/Custom/HubReferralWidget';

<HubReferralWidget 
  title="멀린 패밀리에 친구를 초대하세요!"
  description="초대받은 친구도, 나도 50코인 즉시 지급!"
/>
```
