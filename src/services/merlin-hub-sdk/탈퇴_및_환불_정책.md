# 허브 통합 계정 탈퇴 및 코인 환불 정책

## 1. 개별 앱 탈퇴 원칙
- 허브 생태계는 여러 앱이 하나의 통합 계정(family_users)을 공유합니다.
- 유저가 특정 앱에서 탈퇴할 경우, 해당 앱의 접근 권한만 해제되며 통합 계정의 데이터(코인, 다른 앱 기록)는 보존됩니다.
- 개별 앱의 최초 가입일 히스토리는 pp_join_dates를 통해 영구 보존됩니다.

## 2. 1개월 유예 및 완전 파기 (Soft Delete)
- 유저가 가입한 **모든 앱(최후의 1개 포함)에서 탈퇴**할 경우, 계정은 즉시 삭제되지 않고 **탈퇴 대기 상태(deleted_at 기록)**로 변경됩니다.
- 탈퇴 후 30일 이내 재로그인 시 계정이 원상 복구됩니다.
- 30일이 경과한 계정은 배치 스크립트에 의해 영구 삭제됩니다.

## 3. 코인 환불 및 정산 로직
- 코인 연동 앱(AggroFilter, WhatEat 등) 중 **마지막 남은 코인 앱**에서 탈퇴 시, 잔여 코인 정산 절차가 발동됩니다.
- **환불 금액 계산 (무료 코인 우선 차감 방식)**:
  - 환불 가능 코인 = Min(현재 총 잔여 코인, 누적 유료 결제 코인)
  - 가입 축하금, 추천인 보상 등 무료 지급된 코인은 환불 대상에서 제외되며 탈퇴 시 자동 소멸됩니다.
- 환불 가능 코인이 존재하는 경우, 즉각적인 탈퇴를 차단하고 **고객센터(채널톡)를 통한 정산 요청**을 안내합니다.
