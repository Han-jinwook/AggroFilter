# 🏛️ Merlin Family Hub 기능 스펙 (Core Specs)

## 1. 🔑 인증 시스템 (Identity & Access)
- **OTP 로그인**: 이메일 기반 6자리 일회용 비밀번호 발송 및 검증.
- **JWT 세션**: 발급된 토큰을 통해 패밀리 앱 간 통합 세션 유지.
- **SSO(Single Sign-On)**: `.sundreamer.app` 도메인 쿠키 공유를 통한 자동 로그인 지원.

## 2. 💰 지갑 및 자산 관리 (Wallet System)
- **통합 잔액**: 모든 패밀리 앱에서 공통으로 사용하는 `family_coin_balances`.
- **코인 트랜잭션**: 
  - `DEDUCT`: 앱 서비스 이용 시 코인 차감.
  - `CHARGE`: 결제를 통한 코인 충전.
  - `REWARD`: 환영 보상, 이벤트 참여 등으로 인한 코인 지급.
- **이력 관리**: 모든 증감 내역은 `family_wallet_logs`에 영구 기록.

## 3. 💳 KCP 결제 엔진 (Payment Engine)
- **결제 준비**: `ordr_idxx` 발급 및 KCP 팝업 초기화 데이터 전송.
- **3단계 폴백 승인**: 
  1. HTTP 8090 포트 직접 승인 (고속)
  2. 로우 소켓(Raw Socket) 통신 (보안)
  3. REST API (최종 백업)
- **사후 검증**: PG사 승인 번호(TNO) 대조 및 데이터 무결성 체크 후 코인 즉시 지급.

## 4. 📱 앱 연동 관리 (App Integration)
- **Client ID / Secret**: 각 앱별 고유 인증 키를 통한 API 접근 제어.
- **Scope 관리**: 앱별 허용된 기능 범위(Wallet 읽기, 쓰기 등) 정의.
- **중복 가입 방지**: 최초 가입 앱 기록 및 패밀리 통합 유저 매핑.

## 5. 📬 통합 이메일링 시스템 (Emailing Bridge)
- **네이버 SMTP 메일링**: nodemailer를 활용한 `smtp.naver.com` 연동. HTML 형식 메일을 지원하여 마케팅 및 안내 시 이미지 첨부 가능.
- **트리거 API**: 개별 패밀리 앱 백엔드에서 허브 API(`/api/notification/send`)를 통해 유저별 맞춤 이메일 발송 위임 가능.

## 6. 🛠️ 관리자 도구 (Admin Tools)
- **수동 코인 조정**: 운영자가 유저의 코인을 직접 지급하거나 차감 가능.
- **서버 상태 모니터링**: KCP 게이트웨이 연결 상태 및 서버 아웃바운드 IP 확인.
- **통합 이메일 발송 관리**:
  - 타겟팅 발송: '전체 사용자' 혹은 '특정 패밀리 앱(어그로필터 등) 사용자'를 구분하여 맞춤 이메일 발송.
  - 중요 공지사항(`isNotice: true`) 옵션: 점검 및 긴급 장애 안내 시 사용자의 이메일 알림 수신 동의 여부와 무관하게 100% 강제 이메일 발송 처리.
