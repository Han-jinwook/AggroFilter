📖 AutoMarketer 사용 가이드

🗂 전체 구조

  • 대시보드: 이 화면. 봇 상태 확인, 옵션 제어, 수집/댓글 현황 모니터링
  • 모듈1 — YouTube 수집·분석: 트렌드 영상을 수집해 AggroFilter 메인 앱에 분석 요청
  • 모듈2 — 커뮤니티 타겟: 커뮤니티 사이트(일반/네이버카페) 등록 및 자동 수집 관리
  • 모듈3 — 댓글 자동화: 분석 완료 영상에 유튜브 댓글 + 커뮤니티 댓글 자동 작성

⚙️ 수집 옵션 (모듈1 — YouTube 수집)

파라미터의미기본값권장 범위

X채널 쿨타임 (일) — 동일 채널 재수집 방지73 ~ 14
딜레이분석 요청 간격 (ms) — API Rate Limit 방어50003000 ~ 10000
카테고리별 쿨타임카테고리마다 채널 재수집 주기를 개별 지정 — 비워두면 X일 기본값 적용—1 ~ 365일
  • 변경 후 반드시 💾 저장 버튼을 눌러야 다음 실행에 적용됩니다.
  • 합산 최대 수집량 = N + M×7개 (중복 제거 후)
  • 수집 대상은 업로드 후 12시간 이내 영상으로 한정 (KST 기준, YouTube API 정확한 데이터)
  • 카테고리 쿨타임 예시 — 뉴스/정치: 3일, 엔터테인먼트: 7일 → 해당 카테고리 채널은 그 주기만큼 건너뜀

🤖 자동 스케줄 (모듈별 독립)

모듈스케줄오토모드 위치특이사항
모듈1 YouTube 수집매일 00:00 / 12:00 KST모듈1 탭 좌측 현황카드수집 → 중복제거 → 자막수집 → 분석등록
모듈2 커뮤니티 타겟매일 00:00 / 12:00 KST모듈2 탭 좌측 현황카드타겟 커뮤니티 게시글 수집
  • 각 모듈의 오토모드를 독립적으로 ON/OFF 가능 — 한 모듈만 자동 실행도 가능
  • 오토모드 OFF 시 해당 모듈은 수동 실행만 가능
  • 수동 실행 버튼 클릭 시 저장된 설정값 그대로 즉시 실행
  • 실행 중에는 버튼이 비활성화되며, 완료 시 리스트 자동 새로고침

💬 유튜브 댓글 등급 기준

등급신뢰도 점수댓글 방향
A 70점 이상 (고신뢰) "이 영상 믿을 만하네요" 류 — AggroFilter 링크로 검증 유도
B 40~69점 (중간) "팩트체크 한번 해봤는데..." 류 — 궁금증 유발
C 39점 이하 (저신뢰) "이런 영상 조심하세요" 류 — 어그로 경고 + 링크
  • 댓글에 삽입되는 링크: https://aggrofilter.com/result/{videoId}
  • 댓글 사이 딜레이: 1~2분 랜덤 (스팸 방지)

🎯 모듈2 — 커뮤니티 타겟 관리

  • 구분: 일반 커뮤니티 / 네이버카페 중 선택
  • 커뮤니티명: 식별용 이름 (예: 에펨코리아, 보배드림)
  • URL: 게시판 URL 등록 — 봇이 해당 게시글을 탐색
  • 닉네임 / 아이디 / 비밀번호: 해당 커뮤니티 로그인 정보 (비밀번호는 ••••로 표시)
  • 키워드: 타겟별 탐지 키워드, 쉼표(,) 구분 — 예: 가짜뉴스, 유튜브 낚시, 어그로
  • 범용 키워드: 수집 설정에서 지정 — 모든 커뮤니티에 공통 적용
  • 최근 24시간 이내 게시글만 대상 (오래된 글 스팸 방지)
  • 활성화/비활성화 토글로 타겟별 on/off 제어 가능
  • 댓글 내용은 Gemini AI가 게시글 본문을 읽고 자연스럽게 생성

📂 CSV 일괄 업로드

  • 커뮤니티 탭 우측 상단 📂 CSV 버튼으로 일괄 등록 가능
  • 파일 형식 (헤더 행 포함 시 자동 스킵):
구분,커뮤니티명,URL,닉네임,아이디,비밀번호,키워드(|구분),메모
일반,에펨코리아,https://fmkorea.com/...,닉네임,myid,mypw,가짜뉴스|낚시,유머게시판
네이버카페,예시카페,https://cafe.naver.com/...,닉네임2,id2,pw2,어그로,

🔐 필수 환경변수 (.env)

변수명용도
YOUTUBE_API_KEYYouTube Data API v3 키
DATABASE_URLSupabase PostgreSQL 연결 URL
GEMINI_API_KEY에펨코리아 댓글 AI 생성용
CHROME_USER_DATA_DIR크롬 사용자 데이터 경로 (Playwright 세션 재사용)
MAIN_APP_URLAggroFilter 메인 앱 URL
  • 크롬 경로 확인: 크롬 주소창에 chrome://version/ → 프로필 경로 항목
  • Playwright 첫 실행 전 크롬에서 유튜브 + 에펨코리아 로그인 필수

⚠️ 운영 주의사항

  • 댓글 작성은 Playwright(실제 브라우저)로 동작 — 크롬이 백그라운드에서 열림
  • YouTube/에펨코리아 정책 변경 시 셀렉터 수정 필요 (youtube-commenter.js, fmkorea-commenter.js)
  • Gemini API 무료 한도: 분당 15회 — 에펨코리아 스캔 게시글이 많으면 503 발생 가능 (자동 재시도 없음)
  • 채널 쿨타임(X)을 너무 낮게 설정하면 동일 채널에 반복 댓글 위험
