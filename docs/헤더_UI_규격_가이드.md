# 🏛️ Merlin Family UI Standard: Header Specification & Layout Guidelines
Version: v1.0
Last Updated: 2026-06-06

본 문서는 Merlin Family 서비스군(어그로필터, 금고지기, 썬드리머 등)의 상단 메뉴 헤더 UI 컴포넌트의 표준 레이아웃, 크기 규격 및 정렬 방식에 대한 규정이다. 

---

## 1. 헤더 기본 레이아웃 규격 (Layout Specifications)

모든 Merlin Family 웹 서비스의 헤더는 데스크톱 및 모바일 뷰 모두에서 일관된 높이와 반응형 최대 가로폭 규격을 가져야 한다.

### 1) 높이 (Height) 규정
* **기본 높이**: **`h-20` (80px)**
  * 헤더 전체 컨테이너의 고정 높이는 `80px`로 통일한다.
  * 내부 정렬은 수직 중앙 정렬 (`items-center`)을 사용한다.
  * 하단 테두리는 `border-b border-slate-200`을 사용하여 페이지 본문과 구분한다.
  * 배경은 `bg-white/80 backdrop-blur-xl`과 `sticky top-0 z-50`을 사용하여 스크롤 시 부드럽게 상단에 고정되도록 처리한다.

### 2) 가로폭 (Width) 및 패딩 (Padding) 규정
* **최대 가로폭**: **`max-w-[1200px]`**
  * 헤더 내부 콘텐츠 영역의 가로폭은 최대 `1200px`로 제한한다. (`mx-auto`로 화면 중앙 정렬)
  * 서비스 전체의 일관성 있는 넓이 조절을 위해 기존 `--app-max-width` (예: 800px)를 사용하지 않고, 헤더 영역은 독립적인 **`max-w-[1200px]`**를 지정하여 넓은 화면에서 양측에 정렬되도록 한다.
* **좌우 패딩**: **`px-6 md:px-8`**
  * 모바일 영역에서는 좌우 패딩 `24px (px-6)`, 태블릿 및 데스크톱 영역에서는 `32px (px-8)`을 적용하여 내부 요소들이 화면 끝에 너무 붙지 않게 여백을 확보한다.

## 2. 헤더 내부 정렬 규정 (Alignment Rules)

헤더 내부의 요소들은 **3단 분할 레이아웃(3-Column Layout)** 혹은 **양끝 정렬(Space-Between Layout)**을 기반으로 정확히 수직 중앙에 배치되어야 한다.

```
+-----------------------------------------------------------------------------------+
|  [로고 & 브랜드]                    [중앙 네비게이션]               [우측 위젯 영역]  |
|  (Logo + SubTitle)                 (Nav Menu Items)            (Wallet + Avatar)  |
+-----------------------------------------------------------------------------------+
```

### 1) 좌측: 브랜드 로고 및 서브타이틀 영역
* **구성**: 메인 서비스 로고 이미지와 서비스 대분류 명칭(예: '유튜브')을 배치한다.
* **정렬**: 수평 정렬 `flex items-center gap-3`, 수직 정렬 `items-center`를 적용하여 로고 이미지와 텍스트 베이스라인이 완벽히 대칭을 이루도록 설정한다.

### 2) 중앙: 네비게이션 메뉴 (Navigation Items)
* **구성**: 주요 페이지 이동용 메뉴(예: 'My Page', '분석 Plaza' 등)를 배치한다.
* **정렬**: 
  * 메뉴 버튼 내부의 아이콘과 텍스트가 세로로 균등 배치될 수 있도록 `flex flex-col items-center gap-1`을 적용한다.
  * 여러 메뉴 항목들은 `flex items-center gap-8`로 데스크톱 뷰 기준 여백을 충분히 준다.

### 3) 우측: 유저 상태 및 트랜잭션 영역 (Widget Area)
* **구성**: 지갑 잔액 표시 위젯(코인) 및 Merlin HubAvatar 표준 프로필 위젯을 배치한다.
* **정렬**: 
  * 지갑 위젯과 프로필 위젯을 `flex items-center gap-4`로 묶어 완벽한 수직 중앙 정렬을 제공한다.
  * 로딩 상태이거나 로그인 상태 변경 시 헤더 높이가 흔들리지 않도록 고정 높이와 패딩을 유지한다.

## 3. 어그로필터 (`AggroFilter`) 반영 상세

* **반영 파일**: [AppHeader Component](file:///d:/AggroFilter/components/c-app-header/index.tsx)
* **반영 내용**:
  * 기존 `max-w-[var(--app-max-width)]` (800px)에서 **`max-w-[1200px]`**로 가로폭 규격을 확장하여 양측 밸런스를 개선.
  * 내부 padding을 `px-6 md:px-8`로 확장하고, 로고 영역과 우측 메뉴들 사이의 간격 수직 중앙 정렬을 완벽하게 맞춤.
  * 각 MenuItem 버튼 및 지갑/프로필 컴포넌트의 `align-items`와 높이를 정렬하여 균형 잡힌 심미적 UI 구성.
