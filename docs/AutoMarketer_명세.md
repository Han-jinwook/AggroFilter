# ⚙️ 어그로필터 오토마케터 (AutoMarketer)

## 1. 개요 (Overview)
이 시스템은 어그로필터 메인 서비스와 분리되어, 로컬 환경에서 단독으로 돌아가는 **'자동화 봇(Bot) 및 제어판'**입니다.

### 1.1 아키텍처 구조
- **위치**: `auto-marketer/` 독립 폴더 (Node.js).
- **실행 방식**: 봇 스크립트 + 로컬 웹 서버(`http://localhost:3001`) 대시보드 UI.
- **DB 연동**: 메인 웹앱과 동일 DB 공유. (의뢰자 식별: `user_id = 'bot'`)

## 2. 핵심 섹션별 기능

### 🚀 섹션 1: 트렌딩 수집 (Trending)
- YouTube 인기 급상승 및 카테고리별 상위 영상 자동 수집.
- 24시간 이내 신규 영상 및 기분석 제외 필터링 적용.

### 🎯 섹션 2: 어그로 키워드 사냥 (Keyword Hunting)
- **전술**: "충격", "폭로", "사기" 등 자극적 키워드 기반 검색 수집.
- **오픈소스 AI 사전 필터**: Ollama(Qwen 2.5 7B)를 이용해 비용 0원으로 어그로 점수 사전 산출.
- **유료 분석**: 컷라인(예: 60점) 이상의 영상만 정밀 분석 수행.

### 🕵️ 섹션 3: 요주의 채널 딥다이브 (Deep Dive)
- 어그로 점수가 반복적으로 높은 채널을 '요주의 채널'로 등록.
- 해당 채널의 신규 영상을 집중적으로 추적하여 분석 데이터 확보.

### 🗣️ 섹션 4: 커뮤니티 댓글 및 발화점 감시 (Community)
- 커뮤니티(에펨코리아 등) 핫게시물 내 유튜브 링크 추출 및 우선 분석.
- 분석 결과 기반 자동 댓글 영업 (Playwright 활용).

## 3. 기술 스택 (Tech Specs)
- **Backend**: Express.js
- **UI**: EJS (경량화 지향)
- **Local AI**: Ollama (Qwen 2.5 7B)
- **DB**: Supabase (PostgreSQL)
