# 📝 어그로필터 기능 스펙

## 1. 서비스 개요
- **목표**: 유튜브 영상의 신뢰도 분석 및 어그로 필터링
- **주요 유저**: 유튜브 시청자 및 분석가

## 2. 핵심 기능 리스트
- **[분석 엔진]**: 
  - 사용자가 입력한 유튜브 URL에서 영상 ID 및 자막(Transcript), 메타데이터 추출
  - **[제로대기 비동기 투트랙]**: 빠른 1차 노출(요약/스포일러) + 정밀 2차 분석(신뢰도/평가) 구조.
  - **[입구컷(Filtering) 정책]**: 뉴스, 교육, IT 등 7대 전문 카테고리 외 자동 차단.
- **[코인 시스템 (허브 중앙 관리)]**: 
  - 분석 요청 시 허브 API를 통해 코인 차감 (단가: 동적 과금, 실시간 토큰 기반 계산)
  - 캐싱 로직: 이미 분석된 영상은 차감 없이 즉시 노출.
- **[광고 타임패스 (Ad-Free Timepass)]**:
  - 유튜브 확장팩 아이콘을 통해 정식 과금 분석이 완료되면 **5분간 사이드/바텀 광고 자동 제거**.
  - 만료 시간(`ad_free_until`)은 분석 응답 JSON으로 전달 → 클라이언트 `localStorage`에 저장.
  - 앱 내 단순 클릭(DB 캐시 조회)은 과금 없음 → 광고 디폴트 ON.
  - 게스트(비로그인) 사용자는 타임패스 미부여.
- **[허브 연동 통합 알림 (이메일 전송)]**: 
  - 관심 채널 구독/등급 변화 혹은 영상 분석 완료 시 Merlin Hub SDK를 통해 등록된 유저 이메일로 알림을 발송.
  - 인앱 푸시 및 알림 종 뱃지/드롭다운 UI는 일체 제공하지 않으며, 모든 중요 공지는 이메일을 통해서만 안내됨.
  - 설정 페이지(`p-settings`) 내 이메일 알림 수신 설정을 허브 프로필 카드(`HubProfileCards`) 및 SDK(`useHub().updateNotificationSettings`)와 직접 동기화하여 제어.


## 3. 연동 기술 및 접점
- **유저 식별**: 허브 UUID (`family_users.id`) 단일 사용.
- **인증**: 허브 OTP/JWT 및 SSO 연동.
- **결제**: KCP PG 연동 (썬드림 주식회사).
- **AI 엔진**: Google Gemini 1.5-flash.

## 4. 정책 및 제약조건
- **유튜브 API 쿼터**: 효율적 필터링을 통한 쿼터 절약.
- **환불 규정**: 디지털 재화 특성(7일 이내/미사용 시) 준수.
