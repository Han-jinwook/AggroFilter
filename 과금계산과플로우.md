# 🪙 어그로필터 정밀 동적 과금 시스템 가이드 (v1.5)
> **작성 일시**: 2026-05-12 00:25 (KST)

본 문서는 어그로필터(AggroFilter)의 AI 분석에 적용된 실시간 동적 과금(Dynamic Billing) 로직과 계산 플로우를 정의합니다.

## 1. 과금 철학 (Billing Philosophy)
- **투명성**: AI API의 실제 사용량(토큰 및 검색)에 기반하여 과금합니다.
- **수익성**: API 원가 대비 약 3배의 합리적인 마진을 유지합니다.
- **합리성**: 1C = 1원 기준으로, 대용량 분석도 부담 없는 가격으로 제공합니다.

## 2. 상세 계산 공식 (Calculation Formula)

### [A] 원가 산출 (Raw Cost)
모든 AI 모델과 도구의 사용량을 '코인(C)' 단위로 환산하여 합산합니다.

1.  **AI 토큰 (AI Tokens)**: 
    - **환율**: **10,000 토큰 = 1C** (기존 대비 10배 인하)
    - **대상 모델**: `GPT-4o-mini` (Speed), `Gemini 2.5 Flash` (Full)
    - **계산식**: `(GPT 토큰 + Gemini 토큰) / 10,000`

2.  **구글 뉴스 검색 (Google Search Grounding)**: 
    - **단가**: **5C (고정제)**
    - **계산식**: `구글 검색 횟수 * 5C`

3.  **합계 원가 (Total Raw Cost)**: 
    - **계산식**: `[1번 항목] + [2번 항목]`

### [B] 최종 청구 (Final Charge)
운영 마진을 적용하여 유저의 지갑에서 최종 차감합니다.
- **최종 금액**: `합계 원가 * 3 (마진 Multiplier)`
- **최소 과금**: 분석당 최소 **3C (3원)**를 보장합니다.

## 3. 데이터 흐름 (Data Flow)

1.  **분석 수행**: 각 모델이 사용한 정확한 토큰 수와 검색 쿼리 목록을 취합합니다.
2.  **비용 계산**: `route.ts`에서 **10,000:1** 환율을 적용하여 원가를 산출합니다.
3.  **최종 결제**: Merlin Hub 백엔드에서 3배 마진을 곱해 최종 금액을 차감합니다.

## 4. 과금 예시 (Billing Examples)
- **일반 영상 (검색 없음, 1.2만 토큰)**: `(1.2C + 0C) * 3 = 3.6C (약 4원)` 차감
- **긴 영상 (검색 없음, 7.2만 토큰)**: `(7.2C + 0C) * 3 = 21.6C (약 22원)` 차감
- **뉴스/시사 영상 (검색 1회, 1.5만 토큰)**: `(1.5C + 5C) * 3 = 19.5C (약 20원)` 차감
